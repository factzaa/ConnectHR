# ระบบรับสมัครงาน (PWA + Supabase)

แอปรับสมัครงานแบบ Progressive Web App ใช้ HTML + JavaScript ล้วน ต่อกับฐานข้อมูล **Supabase**
เปิดใช้ได้ทันที ไม่ต้อง build ไม่ต้องลง Node

---

## โครงสร้างโปรเจกต์

```
recruitment-pwa/
├── index.html          # หน้า QR Code / ลิงก์ สำหรับแจกให้ผู้สมัคร
├── apply.html          # หน้าใบสมัคร (PWA ติดตั้งบนมือถือได้)
├── admin.html          # หน้าหลังบ้าน (ต้องล็อกอิน)
├── config.js           # ★ ตั้งค่า Supabase + ชื่อบริษัท + ตำแหน่งงาน
├── manifest.json       # ข้อมูล PWA
├── sw.js               # Service Worker (cache/ออฟไลน์)
├── css/style.css       # สไตล์ทั้งหมด
├── js/
│   ├── supabase-init.js  # สร้าง client กลาง
│   ├── apply.js          # ตรรกะหน้าใบสมัคร
│   └── admin.js          # ตรรกะหน้าหลังบ้าน
├── icons/              # ไอคอน PWA
└── supabase/schema.sql # ★ สคริปต์สร้างตาราง + สิทธิ์ + storage
```

★ = ไฟล์ที่ต้องแก้/รันก่อนใช้งานจริง

---

## ขั้นตอนติดตั้ง (5 ขั้น)

### 1. สร้างโปรเจกต์ Supabase
ไปที่ https://supabase.com → New project → ตั้งชื่อและรหัสผ่านฐานข้อมูล

### 2. สร้างตารางและสิทธิ์
Dashboard → **SQL Editor** → New query → วางเนื้อหาไฟล์ `supabase/schema.sql` ทั้งหมด → กด **Run**
(สคริปต์นี้สร้างตาราง `applicants`, เปิด Row Level Security, และสร้าง Storage bucket ชื่อ `applications` ให้อัตโนมัติ)

### 3. ตั้งค่า `config.js`
คัดลอกค่าจาก Dashboard → **Project Settings → API** มาใส่:

```js
SUPABASE_URL:      "https://xxxx.supabase.co",
SUPABASE_ANON_KEY: "eyJhbGciOi...",   // ใช้ค่า anon / public เท่านั้น
```
พร้อมแก้ชื่อบริษัทและตำแหน่งงานที่เปิดรับได้ในไฟล์เดียวกัน

### 4. สร้างบัญชีผู้ดูแล (แอดมิน)
Dashboard → **Authentication → Users → Add user** → ใส่อีเมล/รหัสผ่าน
บัญชีนี้ใช้ล็อกอินที่หน้า `admin.html`

### 5. นำขึ้นโฮสต์ (ให้ได้ URL จริงสำหรับ QR)
เลือกวิธีใดวิธีหนึ่ง — ทั้งหมดฟรี:
- **Netlify Drop**: ลากโฟลเดอร์ `recruitment-pwa` ทั้งอัน วางที่ https://app.netlify.com/drop
- **Vercel**: `vercel` ในโฟลเดอร์ หรืออัปโหลดผ่านหน้าเว็บ
- **GitHub Pages**: push ขึ้น repo แล้วเปิด Pages

> PWA ต้องรันบน **https** จึงจะติดตั้งบนมือถือได้ (โฮสต์ข้างบนเป็น https อยู่แล้ว)

## การใช้งาน

**ผู้สมัคร** — เปิด `index.html` เพื่อดู/พิมพ์ QR Code หรือส่งลิงก์ `apply.html` ให้โดยตรง
ผู้สมัครกรอกข้อมูล อัปโหลดรูปถ่าย และแนบเอกสารแยกชนิด (บัตรประชาชน, ทะเบียนบ้าน, วุฒิการศึกษา, ใบอนุญาตทำงาน, และอื่น ๆ แนบได้หลายไฟล์) โดยเลือกได้ทั้งแนบไฟล์ หรือถ่ายรูป/สแกนจากกล้องมือถือทันที แล้วกดส่ง — ข้อมูลเข้า Supabase ทันที

**ผู้ดูแล** — เปิด `admin.html` → ล็อกอิน → ดูตารางใบสมัคร ค้นหา กรองตามสถานะ/ตำแหน่ง
คลิกแถวเพื่อดูรายละเอียด เปลี่ยนสถานะ (ใหม่ → พิจารณา → สัมภาษณ์ → ผ่าน/ไม่ผ่าน)
บันทึกโน้ต และกด **Export CSV** เพื่อดาวน์โหลด (เปิดใน Excel ได้ รองรับภาษาไทย)

---

## ทดสอบบนเครื่องก่อนขึ้นจริง
เปิดในโฟลเดอร์แล้วรันเซิร์ฟเวอร์ง่าย ๆ:
```bash
python -m http.server 8000
# เปิด http://localhost:8000
```
(เปิดไฟล์ตรง ๆ ด้วย file:// ตัว Service Worker และการอัปโหลดบางส่วนอาจไม่ทำงาน — แนะนำผ่านเซิร์ฟเวอร์)

---

## หมายเหตุความปลอดภัย
- ใช้ **anon key** เท่านั้นในฝั่งหน้าเว็บ — ปลอดภัยเพราะสิทธิ์ถูกคุมด้วย Row Level Security ในฐานข้อมูล
- ผู้ที่ไม่ล็อกอิน: เพิ่มใบสมัครได้อย่างเดียว **อ่าน/แก้/ลบไม่ได้**
- เฉพาะแอดมินที่ล็อกอินเท่านั้นที่เห็นข้อมูลและไฟล์แนบ (ไฟล์ให้ผ่าน signed URL อายุ 1 ชม.)
